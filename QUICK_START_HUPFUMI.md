## üîß API Quick Reference

### Complete District Profile
```bash
curl "http://localhost:8000/api/district/Bindura/complete-profile"
```

Returns: Geographic info, crops, markets, profitability, challenges, opportunities

### Ask District-Specific Question
```bash
curl -X POST "http://localhost:8000/api/district/Chimanimani/ask?question=What%20crops%20grow%20here"
```

Returns: AI-generated answer with citations

### Get Market Information
```bash
curl "http://localhost:8000/api/district/Masvingo/markets"
```

Returns: Local markets, prices, where to sell

### Calculate Crop Profitability
```bash
curl "http://localhost:8000/advisory/margin/maize/Bindura"
```

Returns: Gross profit margin, costs, revenue, breakeven analysis

### Compare Crops
```bash
curl "http://localhost:8000/api/district/Harare/profitability-comparison"
```

Returns: Top profitable crops ranked for that district
